2017-12-10 Python Python

doctestを触ってみた

今日の内容は、Pythonでテストしたいにまとめている内容の詳細部分です。

doctestを触ってみました。

sample.py

"""
値を1増加させる関数です
>>> add_one(1)
2
"""


def add_one(x):
    """
    値が1増加します
    >>> [add_one(x) for x in range(5)]
    [1, 2, 3, 4, 5]
    """
    return x + 1


if __name__ == '__main__':
    import doctest
    doctest.testmod()

実行する

$ python sample.py

正常に動作した場合はなにも表示されない

sample.pyの14行目を return x + 2 と書き換えて実行

$ python sample.py
