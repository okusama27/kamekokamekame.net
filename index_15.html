2015-05-28 Book Book

How Google Works

How Google Works (ハウ・グーグル・ワークス) ―私たちの働き方とマネジメント

スーパーサイヤ人の集め方といった感じの本でした。

孔子の言葉に「己に如かざる者を友とするなかれ」というのがありますが、みんな心の中で思っている「いけている人とだけ働きたい」という思いをどうしたら実践できるか。Googleはどう実践してきたかというのの解説でした。

ある程度のスーパーエンジニアが集まれば、あとは加速度的に集まってくると思いますが、少人数だった時にどうやってスーパーエンジニアを集めたかが全然書いてなかったです。どうやって人脈を作って、どうやって入社してもらったのか？ある程度の規模になってからスーパーエンジニアがいっぱい受けにきたけど、変な人入れるくらいだったら、いけている人を見落としてもいい。とか、全然参考にならない！選べる立場でいいですね。くらいの感想です。

2015-04-15 Python Python

もう一回virtualenvの使い方

virtualenvの使い方を習得できていないというか、適当に検索して使ってみたけど、皆自分に都合がいいことしか書いてないのでわかりづらい。初心者に優しくない！

あと、余計なツールは逆にめんどくさい！！と思う。 ということで、普通にvirtualenvのドキュメントを読みました。そうだ！私って英語読めるんだった！！

インストール

$ [sudo] pip install virtualenv

作成

$ virtualenv ENV

アクティブ化

$ source ENV/bin/activate

アクティブ解除

$ deactivate

eggを作ってみる

Pythonのモジュールの配布方法としてeggというzipファイルを作成して配るらしい。

完全にこちらのサイトを参考にしました。ありがとうございます。mm

python開発環境の構築から作成したeggを別環境へインストールするまで pasterのインストール

$ easy_install PasteScript

雛形の作成

$ paster create -t basic_package MyPkg

これやると色々聞かれる。バージョンとか作者の名前とか。聞かれたことはsetup.py中に記録されています。
