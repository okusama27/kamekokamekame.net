Hello Go¶

2015-07-05

Goって書いたことないので、やってみます。

公式ドキュメントみっけ

Mac用のインストーラがあったので、インストール、チュートリアル

hello.go

prepackage main
import "fmt"
func main() {
     fmt.Println("Hello, Go")
}

実行

$ go run hello.go
Hello, Go

でた。

初めてのGo。終わり。
