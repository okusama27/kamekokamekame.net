virtualenv使ってみる

独立した環境を利用するためのツールがある。virtualenvってやつで、Python3からはpyvenv(venv?)ってのを利用するらしい。

で、virtualenvを利用しやすくするものにpyenvとか、なんちゃらenvとかいろいろあるみたい。Goで書かれたdirenvってのもぐいぐい来ているらしいが、また今度。 とりあえずインストール

$ sudo pip install virtualenv

virtualenvwrapperってのを入れると環境の切り替えが楽らしい

sudo pip install virtualenvwrapper

環境設定 .bashrcに追加

#virtualenv settings
export WORKON_HOME=$HOME/.virtualenvs
. /Library/Frameworks/Python.framework/Versions/2.7/bin/virtualenvwrapper.sh

環境のファイル作る

mkvirtualenv &#123;env name&#125;

作った環境一覧

workon

とある環境をアクティブにする

workon &#123;env name&#125;

アクティブ終了

deactivate

削除するとき

rmvirtualenv &#123;env name&#125;

私の環境はMacOSXです。
